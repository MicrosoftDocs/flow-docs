# Power Automate Desktop (Preview) 소개

[!INCLUDE [cc-beta-prerelease-disclaimer.md](../../includes/cc-beta-prerelease-disclaimer.md)]

Power Automate Desktop은 기존의 로봇형 프로세스 자동화(RPA, Robotic Process Automation)를 확장시켜 줍니다. UI 흐름의 일부로서 모든 반복적인 데스크톱 프로세스를 자동화 하실 수 있게 됩니다. 미리 만들어둔 끓어 놓기(drag-and-drop) 가능한 작업들을 활용하시건 직접 자신의 UI 흐름에서 실행한 내용을 녹화하실 수 있습니다.

새로운 직관적인 Power Automate Desktop으로 더 빠르고 손쉽게 자동화 하실 수 있습니다. 파일, 폴더, 엑셀 그리고 데스크톱 자동화 기능들을 포함한 폭 넓은 작업들을 새로운 흐름 디자이너로 수행해보세요.

Power Automate Desktop은 터미널 에뮬레이터, 현대형 웹 그리고 데스크톱 응용프로그램과 같은 레거시 응용프로그램과 엑셀파일과 폴더 등을 통한 자동화를 가능하게 합니다. 응용프로그램 UI 요소, 이미지 또는 좌표 등을 사용해 컴퓨터와 상호작용 해보세요.
